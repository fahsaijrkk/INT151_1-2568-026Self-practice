ข้อ 1: ใช้ substring(start, end) เพื่อดึงตัวอักษรบางตัวจากสตริง เริ่มที่ตำแหน่ง i และดึง n ตัวอักษร เช่น 'thisisastring' เริ่มที่ i = 7 ดึง 3 ตัวอักษร → 'tri'

ข้อ 2: ใช้ indexOf(substring) เพื่อหาตำแหน่งของสตริงย่อยในสตริงหลัก ถ้าเจอจะคืนตำแหน่งตัวแรก ถ้าไม่เจอคืน -1 เช่น 'is blue' ใน 'sky is blue' → เริ่มที่ตำแหน่ง 4

ข้อ 3: แปลงตัวอักษรบางส่วนของสตริงเป็นตัวพิมพ์ใหญ่ โดยใช้ substring แบ่งสตริงออกเป็น 3 ส่วน: ก่อนช่วงที่ต้องการ, ช่วงที่ต้องการแปลงเป็น uppercase, หลังช่วงนั้น แล้วเอามาต่อกัน เช่น 'this is a very very long string ' จาก i = 5 ถึง j = 7 → 'this IS a very very long string '

ข้อ 4: ลบองค์ประกอบของอาร์เรย์จากตำแหน่ง i ถึง j (exclusive) โดยใช้ splice(start, count) ซึ่ง count = j - i เช่น ['a', 'b', 'c', 'd', 'e'] ลบจากตำแหน่ง 2 ถึง 4 → เหลือ ['a', 'b', 'e']
